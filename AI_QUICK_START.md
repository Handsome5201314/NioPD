## 🎯 三步开始使用

### 第1步: 配置API密钥

1. 打开浏览器访问 http://localhost:5173/
2. 登录您的账号
3. 进入 **"⚙️ 模型配置"** 页面
4. 填写您的API配置：

```
API提供商: siliconflow
API端点: https://api.siliconflow.cn/v1
API密钥: sk-mmmpjonhbtofrpudilhndvpvzfltiplyulzwcyoanyxsgzoy
模型名称: Qwen/Qwen2.5-7B-Instruct
Temperature: 0.7
最大Token: 2000
```

5. 点击 **"保存配置"**

**注意**: 您的API密钥已在医生模拟器中配置过，可直接使用。

### 第2步: 使用智能工作台

1. 进入 **"🚀 智能工作台"**
2. 在输入框输入您的需求，例如：

```
我想做一个健身减肥APP，需要考虑哪些产品功能和技术架构？
```

3. 点击 **"发送"** 或按 **Enter**

### 第3步: 查看专家分析

- **左侧对话区**: 查看nio综合后的最终建议
- **右侧专家面板**: 查看各专家的详细分析
- **专家卡片**: 显示专家头像、角色、核心建议

## 🛠️ 自定义专家

### 添加自定义专家

使用浏览器控制台（F12）执行：

```javascript
const token = localStorage.getItem('userToken')

fetch('/api/ai/experts/custom', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`
  },
  body: JSON.stringify({
    id: 'marketing-expert',
    name: 'marketing-expert',
    role: '市场营销专家',
    systemPrompt: '你是破界实验室的市场营销专家，擅长品牌定位、增长策略、内容营销。\n\n你的专长：\n1. 品牌定位：目标用户、品牌形象、差异化\n2. 增长策略：获客渠道、转化优化\n3. 内容营销：SEO/SEM、社交媒体\n\n回复要点：基于数据的营销策略、具体的渠道建议、可衡量的目标。',
    expertiseAreas: ['品牌定位', '增长策略', '内容营销'],
    triggerKeywords: ['营销', '推广', '品牌', '获客']
  })
})
.then(res => res.json())
.then(data => console.log('添加成功:', data))
```

### 查看所有专家

```javascript
fetch('/api/ai/experts')
  .then(res => res.json())
  .then(data => console.log('专家列表:', data))
```

## ⚠️ 常见问题

### 问题1: 发送消息后显示"API配置"错误

**原因**: 未设置API密钥

**解决**:
1. 进入"模型配置"页面
2. 填写API端点和密钥
3. 点击"保存配置"

### 问题2: AI回复显示"网络错误"

**原因**:
- 后端服务未启动
- 网络连接问题
- API密钥无效

**解决**:
1. 检查后端是否运行：访问 http://localhost:3001/api/health
2. 查看后端日志：在终端查看输出
3. 验证API密钥是否正确

### 问题3: 专家回复很慢

**原因**: AI模型推理需要时间

**正常情况**:
- nio编排：1-2秒
- 3个专家并行分析：3-5秒
- nio综合：1-2秒
- 总耗时：5-10秒

**优化建议**:
- 使用更快的模型（如Qwen2.5-3B）
- 减少maxTokens（如1000）
- 启用缓存功能
